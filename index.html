Privacy Policy for Flashscore Web Insights

Last Updated: December 3, 2025

Introduction

Flashscore Web Insights ("the Extension") is a Chrome browser extension developed by Marek Kerka. This privacy policy explains how the Extension collects, uses, and protects your information when you use our service.

Developer Information

Developer: Marek Kerka

Contact Email: marek.kerka@gmail.com

Extension Name: Flashscore Web Insights

Version: 1.8.1

Information We Collect

Data Collected Automatically

When you use the Extension, we automatically collect:

  • Page URLs: The URLs of FlashScore pages you analyze (match pages, team pages)
  • Page Content: The text content from FlashScore pages that you choose to analyze
  • Browser Information: Basic browser information needed for the extension to function
  • Authentication Cookies: Session cookies for authenticated access to the API service

Data You Provide

  • Language Preference: Your selected language for analysis results
  • Questions: Questions you submit through the interactive Q&A feature

How We Use Your Information

We use the collected information for the following purposes:

  1. Content Analysis: To provide AI-powered analysis and insights of FlashScore pages
  2. Service Improvement: To improve the quality and accuracy of our analysis
  3. Feature Functionality: To enable interactive Q&A and other extension features
  4. Caching: To temporarily cache analysis results for better performance (5 minutes)

Data Sharing and Third Parties

OpenAI Integration

The Extension sends page content to OpenAI's API for AI-powered analysis. This data sharing is essential for the core functionality of the Extension. OpenAI's privacy policy can be found at: https://openai.com/policies/privacy-policy

What is sent to OpenAI:

  • Text content from FlashScore pages
  • Your questions submitted through the Q&A feature
  • Language preference for responses

What is NOT sent to OpenAI:

  • Personal identification information
  • Browser history
  • Data from websites other than FlashScore

No Other Third Parties

We do not share your data with any third parties other than OpenAI for analysis purposes. We do not sell, rent, or trade your information.

Data Storage and Retention

Temporary Caching

  • Analysis results are cached temporarily (5 minutes) on our backend server
  • Cache is automatically cleared after expiration
  • No long-term storage of analyzed content

No User Profiles

  • We do not create user profiles
  • We do not track your browsing history
  • We do not store personal identification information

Server Logs

  • Temporary server logs may be created for debugging and error tracking
  • Logs do not contain personally identifiable information
  • Logs are automatically deleted on a regular basis

Permissions Explained

The Extension requests the following Chrome permissions:

  • activeTab: To access the current FlashScore page URL when you use the extension
  • sidePanel: To display the analysis results in a side panel interface
  • host_permissions (flashscore.com): To inject content scripts only on FlashScore pages

These permissions are used solely for the Extension's core functionality and are not used for tracking or data collection beyond what is necessary.

Data Security

We implement security measures to protect your information:

  • HTTPS/TLS: All communication between the Extension and our API uses encrypted connections
  • Authentication: Cookie-based authentication to prevent unauthorized API access
  • No Storage: We do not permanently store analyzed content or personal data
  • Limited Access: API access is restricted to authenticated requests from the Extension

Your Rights and Choices

You have the following rights:

  • Use Control: You control when the Extension analyzes pages by manually triggering analysis
  • Uninstall: You can uninstall the Extension at any time through Chrome's extension settings
  • Data Access: Since we don't store personal data, there is no persistent data to access or delete
  • Questions: Contact us at marek.kerka@gmail.com for any privacy-related questions

Children's Privacy

The Extension is not directed at children under the age of 13. We do not knowingly collect information from children under 13. If you believe we have collected information from a child under 13, please contact us immediately.

Significant changes will be communicated through:

  • Extension update notes
  • Notification in the Extension interface (if applicable)
